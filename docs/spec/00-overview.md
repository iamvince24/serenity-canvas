# 專案概述 (Overview)

**專案名稱：Serenity Canvas**

這是一個基於網頁的視覺化白板工具（Web App），旨在解決 Obsidian Canvas 在「素材蒐集」與「內容排版」上的痛點。

Serenity Canvas 是一個基於網頁的視覺化白板工具，使用者透過瀏覽器外掛快速剪藏網頁內容，在此 Web App 中進行視覺化整理、編輯與關聯，最終將整理好的成果（包含 Markdown 筆記與 Canvas 佈局檔）一鍵匯出至 Obsidian。

## 核心價值

1. **完整內容顯示**：解決 Obsidian 卡片在縮放時隱藏內文的問題，提供更像 Heptabase 的閱讀體驗。
2. **無縫剪藏**：透過 Chrome Extension 直接將網頁靈感轉化為白板卡片。
3. **原生相容**：匯出的檔案完全符合 Obsidian .canvas 與 .md 格式標準。
