# Serenity Canvas

> 一個優雅的視覺化白板工具，作為 Obsidian 的素材收集空間。

## 專案願景

Serenity Canvas 旨在創造一個寧靜、流暢的思考環境，讓使用者能夠輕鬆收集、整理和視覺化複雜想法，最終無縫匯出至 Obsidian。

## 核心價值

- **完整內容顯示**：解決 Obsidian Canvas 在縮放時隱藏內文的問題
- **無縫剪藏**：透過 Chrome Extension 直接將網頁靈感轉化為白板卡片
- **原生相容**：匯出的檔案完全符合 Obsidian .canvas 與 .md 格式標準
